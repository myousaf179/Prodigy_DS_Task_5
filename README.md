# Traffic Accident Analysis

## Project Overview
This project involves analyzing traffic accident data to identify patterns and factors contributing to accidents. As part of my experience at Prodigy InfoTech, I completed a comprehensive analysis, focusing on different factors such as time of day, weather conditions, and road conditions.
